wordpress.org / museum

Eight ways to see a history

Every WordPress release from 2003 to today, rendered as seven wildly different exhibits. Pick a door.

Exhibit I

Desktop Museum

Boot into a Windows XP desktop where every version is a shortcut, the Start menu is a time machine, and the taskbar keeps ticking.
Exhibit II

Winamp Timeline

A skeumorphic music player where every release is a track. Cue up 2003, drop the needle, and the Playground picks up the beat.
Exhibit III

Scrapbook Time Machine

A handmade scrapbook you scroll through — polaroids that develop into live Playgrounds, handwritten margin notes, wax seals, coffee rings.
Exhibit IV

Kubrick Weblog

A period-accurate early-WordPress blog where every release is a post. Click the Dashboard to open a working fake wp-admin: draggable widgets, color schemes, QuickPress, and the Hello Dolly plugin.
Exhibit V · scrollytelling

Horizontal Magnetic

Scroll left-to-right through fifty-three full-viewport slides. Parallax portraits, chiptune that shifts key per decade, and a ribbon that snaps to any release.
Exhibit VI · scrollytelling

Pinned Storytelling

Each release pins to the screen while it unfolds in four beats — portrait drawing in, trivia surfacing, screenshot breathing, Playground booting. Music layers features as the product does.
Exhibit VII · 3D

The Jazz Corridor

A first-person walk through a 3D CSS museum. Every frame a release, every plaque a codename portrait. Step into any painting and the Playground fills the room.
Exhibit VIII · Deep Dive

The Impossible Museum

The corridor breaks open. You jump, fall, shrink, walk the ceiling, and drift through nine releases as gravity and scale rewrite themselves between rooms. Each stop zooms in — pixel portraits, lost screenshots, deeper trivia.
Curated with care. Each exhibit loads live WordPress instances through WordPress Playground.
